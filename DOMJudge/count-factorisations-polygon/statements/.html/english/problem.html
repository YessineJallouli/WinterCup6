Count Factorisations
time limit per test
2 seconds
memory limit per test
256 megabytes
input
standard input
output
standard output

Given a number $$$n.$$$ A factorisation of $$$n$$$ is a multi-set of integers $$$\{ d_1 \dots d_r \}$$$ whose elements multiplies to $$$n:$$$ $$$$$$ n=\prod_{i=1}^r d_i $$$$$$

Given $$$n,$$$ Find the number of its possible factorisations.

Input

A line containing one integer $$$1 \le n \le 10^9$$$

Output

An integer representing the number of possible factorisations.

Examples

Input
1800
Output
137
Input
340510170
Output
21147

Note

Two multi-sets are equal if every element occurs the same number of times in both sets.
